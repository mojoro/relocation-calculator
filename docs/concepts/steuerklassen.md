# Steuerklassen (Tax Classes)

Germany's six Steuerklassen (tax classes I–VI) determine the withholding tax rate applied to wages each month. Class I: single with no special circumstances. Class II: single parent. Class III: married, higher-earning spouse (paired with Class V for the other). Class IV: married, roughly equal incomes. Class V: married, lower-earning spouse (paired with Class III). Class VI: second job or additional employment. Tax class affects the monthly net salary but not the final annual tax owed — any differences are settled in the annual Steuererklärung (tax return).
