# Tailwind CSS

Tailwind CSS is a utility-first CSS framework where you compose styles by applying small single-purpose class names directly in HTML (`flex`, `items-center`, `gap-4`, `text-sm`, `rounded-lg`). There is no pre-built component library — you build components from primitives. Tailwind 4 uses CSS-native configuration via `@import "tailwindcss"` in a CSS file rather than a JavaScript config file. It works alongside CSS custom properties — tokens define semantic values, Tailwind utilities handle layout, spacing, and typography.
