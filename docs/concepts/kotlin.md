# Kotlin

Kotlin is a statically typed JVM language developed by JetBrains, now the recommended language for Android and a first-class Spring Boot language. It compiles to Java bytecode, interoperates fully with Java, and adds data classes, extension functions, null safety (`String?` vs `String`), coroutines, and concise syntax. Key differences from Java: no checked exceptions, `val`/`var` instead of `final`, trailing lambdas, and string templates. Spring Boot 3 has first-class Kotlin support via the `kotlin-spring` and `kotlin-jpa` Gradle plugins.
