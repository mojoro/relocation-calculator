# Angular

Angular is a TypeScript-based web application framework maintained by Google. It uses components, services, and modules to build single-page applications. Angular 17+ embraces standalone components (no NgModules), Signals for reactivity, and built-in SSR support. It provides a complete solution including a router, HTTP client, reactive forms, and a CLI — making it a full framework rather than just a UI library.
