# Spring Boot

Spring Boot is a convention-over-configuration framework for building production-ready JVM applications. It provides auto-configuration (no XML required), an embedded Tomcat/Undertow server, and starter dependencies for common tasks. Key annotations: `@RestController` (marks a class as an HTTP handler that returns JSON), `@Service` (marks a class as a business logic bean), `@RequestBody` (deserializes the JSON request body), `@Valid` (triggers Bean Validation on the deserialized object). Spring Boot 3 requires JDK 17+ and is built on Spring Framework 6.
